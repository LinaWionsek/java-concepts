Was ist eine IDE?
	Ein Betriebssystem für Entwickler.
	Ein Programm oder eine Sammlung von Programmen, mit denen programmiert wird.
	Eine Datenbank zur Speicherung von Code.
	Ein Compiler für maschinensprachebasierte Programme.

Was sind Variablen?
	Dateien, die Code enthalten.
	Funktionen, die keine Eingaben benötigen.
	Variablen haben einen Bezeichner, unter dem sie einen Wert von einem bestimmten Typ speichern.
	Programme, die automatisch ausgeführt werden.

Was ist ein Zustand (State) in der Programmierung?
	Ein Fehler im Code.
	Daten ergeben ein gemeinsames Bild, und das hat eine besondere Bedeutung.
	Eine Programmiersprache.
	Ein Speicherort für Variablen.

Was versteht man unter einer Zuweisung?
	Das Umbenennen eines Programms.
	Das Starten eines Algorithmus.
	Das Löschen einer Variable.
	Eine Anweisung, in der einer Variable ein Wert zugewiesen wird.

Was ist eine Verzweigung?
	Eine Art von Schleife.
	Eine Methode zur Datenkompression.
	Wenn ein bestimmter Zustand besteht, dann soll diese Anweisung passieren, sonst diese.
	Eine Speicherstruktur für große Datenmengen.

Was ist UML?
	Ein Betriebssystem für Entwickler.
	Ein Dateiformat für Quellcode.
	Eine Programmiersprache für künstliche Intelligenz.
	Modellierungsformen, die zum Dokumentieren und Kommunizieren verwendet werden kann.
